## How to start
- installing Python
  
- installing required Python modules
```bash
pip install inference
pip install python-dotenv
```
File paths and environment variables are managed in ".env"

- setting up API_KEY
```bash
export API_KEY=<your api key>
```

- run the Python script
```bash
python app.py
```
